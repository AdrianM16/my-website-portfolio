# Adrian M. Anunciacion — Portfolio Website (v2)

This is my personal portfolio website built with **HTML, CSS, and JavaScript**.  
It highlights my profile, skills, projects, resume, and contact details.

## Pages
- **Home**: quick intro + links to main sections
- **About**: short background, skills, soft skills, and certifications preview (with modal)
- **Projects**: project cards with screenshots and links
- **Resume**: embedded resume preview/download
- **Contact**: contact details and page layout for reaching out

## Features
- **Responsive layout** (desktop / tablet / mobile)
- **Light / Dark mode toggle**
- **Simple scroll animations**
- **Certifications gallery** with a preview modal

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)

## Folder Structure (main)
- `assets/` images, project screenshots, and certificates
- `css/` site styles
- `js/` scripts
- `*.html` pages

## How to Run
1. Download or clone the project.
2. Open `index.html` in your browser.

Tip: If you want live reload, you can use VS Code **Live Server**.

## Notes
- All styling for the About page is scoped using `.page-about` to avoid affecting other pages.
